## 1. Project GitHub URL
This repository contains both the **Frontend** and **Backend** code.

**Repository URL**: `https://github.com/DarusSalamSakib/news-portal-backend`

### Directory Structure
- `/` (Root): Contains Frontend HTML/CSS/JS files.
- `/news-portal-backend`: Contains Node.js/Express Backend code.

### Setup Instructions

#### Backend
1. Navigate to the backend directory: `cd news-portal-backend`
2. Install dependencies: `npm install`
3. Start the server (runs on port 3000): `node server.js`

#### Frontend
1. Open `index.html` in your browser.
2. Ensure the backend is running on port 3000.
